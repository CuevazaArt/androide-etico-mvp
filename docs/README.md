# Documentación

Este directorio contiene la documentación técnica y conceptual del proyecto.

## Archivos

- `Androide_Etico_Analisis_Integral_v3.docx` — Documento completo del modelo
  con notas explicativas, críticas, simulaciones y evaluación de viabilidad.
  Basado en el análisis de 40+ documentos del acervo del proyecto.

## Versión actual: v4.0 (Marzo 2026)

La v4 incorpora la **capa de lenguaje natural** (`llm_layer.py`):
- Percepción de situaciones en texto libre → señales numéricas
- Comunicación verbal del androide con tono, gestos HAX y voz en off
- Moralejas narrativas expandidas desde cada polo ético
- Principio de diseño: **el LLM no decide, el kernel decide**

Ver `CHANGELOG.md` para el historial completo de versiones.

## Para contribuyentes
